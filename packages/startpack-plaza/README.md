# `@zeus/startpack-plaza`

Start pack narrativo mínimo del mundo A (WP-U111): story-board
(`solve-inline` / actos) + labelset + línea + casos. Parametrizable desde
el editor; no es un juego de producto (delta/pozo/solve-coagula).

```bash
npm run release:startpack -- --game plaza
npm install ./zeus-startpack-plaza-0.1.0.tgz
```

Loader: `loadStartPack()` → `{ gamemap, storyBoard, labelset, casosMd, volumesRoot, actaPath, env }`.
