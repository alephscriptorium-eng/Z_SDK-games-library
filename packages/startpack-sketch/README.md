# `@zeus/startpack-sketch`

Start pack mínimo del mundo A (WP-U70): escena + labelset + línea + casos.
Parametrizable desde el editor; no es un juego de producto (delta/pozo).

```bash
npm run release:startpack -- --game sketch
npm install ./zeus-startpack-sketch-0.1.0.tgz
```

Loader: `loadStartPack()` → `{ gamemap, scene, labelset, casosMd, volumesRoot, actaPath, env }`.
